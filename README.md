# CHIP-8 emulator
it is a very simple CHIP-8 emualtor unnecessarily bloated with the c++23.
## Building
just run 
```sh
$ make install 
```
## Usage
run using the command line.
example of running PONG game:
```sh
$ ./chip8 roms/PONG
```
